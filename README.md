# bd-copy
Una extensión para Firefox diseñada para optimizar el flujo de estudio en la plataforma boot.dev.

Permite copiar automáticamente al portapapeles la información clave de la lección actual: nombre del curso, capítulo, título de la lección y su contenido textual completo. Elimina la necesidad de seleccionar y copiar el texto manualmente.

Su propósito es agilizar la transferencia de material educativo hacia aplicaciones de toma de notas, bases de datos personales o herramientas de inteligencia artificial para generar resúmenes y contexto.
